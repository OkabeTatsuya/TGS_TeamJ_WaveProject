なみにのれ

開発した「なみにのれ」のプロジェクトデータです。

「ProjectData」フォルダの「GameSources」の中にソースコードが入っています。

「なみにのれ_実行データ」フォルダの中の
「なみにのれ.exe」を起動するとゲームを実行することができます。
